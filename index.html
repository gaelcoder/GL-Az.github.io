Website: Loja de Discos

Um pequeno projeto single-page que representa uma loja de discos. Nada é verdadeiramente funcional, serviu apenas para treino de HTML, CSS e JavaScript (nossa trindade do front-end).

API: Análise de Sensores

Projeto de uma API que recebe dados de sensores por entrada em um formulário ou por upload de um arquivo CSV. E, a partir deles, exibe em um gráfico a média de valores em: 24 horas, 48 horas, 7 dias e 1 mês. Desenvolvido com Python, Django, HTML, CSS e JavaScript.

API: Gerenciamento de Habitats

API desenvolvida a partir de um desafio proposto por uma vaga de estágio. Ela é capaz de classificar e encontrar habitats pros animais do zoológico baseado em dados como: espaço ocupado, se são sociais, se estão na base de dados e etc. Desenvolvida puramente em JavaScript.
